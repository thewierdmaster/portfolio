Projects

Aqua Vision - Tackling Plastic Pollution with AI

Description

Aqua Vision is an end-to-end solution created to tackle plastic pollution, utilizing advanced state-of-the-art models like YOLOv8, Faster R-CNN, and YOLONAS to detect and promptly notify nearby government agencies and NGOs.

    • Users upload images or capture photos/videos for real-time plastic object detection, using models like Faster R-CNN, YOLOv8, YOLO-NAS, and DETR. Predictions trigger notifications to nearby NGOs.

    • Detect and Discover:

      • Users upload images or capture photos/videos for real-time plastic object detection, using models like Faster R-CNN, YOLOv8, YOLO-NAS, and DETR.
      • Predictions trigger notifications to nearby NGOs.

    • Aqua Guardians:

      • Detected plastic objects are accessible with counts, geotags, and images.
      • NGOs manage requests, marking them as completed, while regular users view pending and completed requests.

    • Aqua Insights Hub:

      • Offers data visualizations and heatmaps for tracking plastic pollution trends by area, category, month, and time.

    • Aqua Allies:

      • A community-driven space where users collaborate, share experiences, and raise awareness about plastic pollution.

    • Object Detection and Computer Vision:

      • AquaVision's object detection begins with a diverse dataset of 3,200 images showing water bodies contaminated with plastics. Handpicked high-quality images were annotated, resulting in a training dataset of 10,000+ images. Four models, including Faster RCNN, YOLOv8, YOLO-NAS, and DETR, were fine-tuned for peak performance.
